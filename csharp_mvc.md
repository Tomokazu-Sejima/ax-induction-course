### REST API とは？

REST API では、HTTP で定義する GET、POST、PUT、DELETE などのリクエストでデータを操作します。

| CRUD 名 | メソッド | 意味     |
| ------- | -------- | -------- |
| Create  | POST     | 作成     |
| Read    | GET      | 読み込み |
| Update  | PUT      | 更新     |
| Delete  | DELETE   | 削除     |
